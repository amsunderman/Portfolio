Project Description

Iowa State’s Office of Sustainability was looking to design an application to act as a medium for Iowa State students to connect to buy and sell used items. The application needs to be easily accessible on all operating systems, implement a clean user interface, and be easy to use. The project also places high value on clean documentation, maintainability, and extensibility to allow the Government of the Student Body to easily manage and maintain the site long term.

Design

        We are still early in the design phase of the project, however, we have been able to start the design process efficiently do to a strong effort to maintain weekly communication with the customer. This has allowed us to generate very detailed requirements that will allow us to make smart and efficient design choices. We have laid out the initial design of our project, choosing to  utilize Laravel (PHP framework) to implement the application on the server to manage clients’ requests for dynamic html pages. We will utilize Iowa State’s NetID login system to allow for a strong sense of community within the website users. More information can be found in the complete design document included below.
